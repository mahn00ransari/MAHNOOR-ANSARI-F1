Function kya hota hai?
<br>
Function JavaScript mein code ka ek block hota hai jo ek specific kaam karta hai. Isay ek baar likhte hain aur jab zarurat ho, usay baar baar chala (call) sakte hain.
<br>
Real-life Example 
<br>
Socho tumhare ghar mein juice machine hai.
<br>
Tum fruit dete ho (input).
<br>
Machine juice banati hai (work).
<br>
Juice mil jata hai (output).
<br>

JavaScript mein function bhi isi tarah kaam karta hai.
<br>
Simple Example
<br>
function sayHello() {
    console.log("Hello!");
}
<br>
sayHello();
<br>

function JavaScript ko batata hai ke function bana rahe hain.
<br>
sayHello ik Function ka nam hy
<br>
{ } Iske andar jo code hoga, woh chalega.
<br>
sayHello(); Function ko call karna, yani usay chalana.
<br>

Output
<br>
Hello!
